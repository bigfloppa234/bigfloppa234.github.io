# Unofficial TrollInstaller source for AltStore

https://bigfloppa234.github.io

How to install:
1. Go to https://bigfloppa234.github.io on your mobile device (Make sure an AltStore beta is installed)
2. Tap on the blue button
3. Allow your browser to open AltStore
4. Press Add Source whenever AltStore asks to add the source


TrollStore can be found at the following repo: https://github.com/opa334/TrollStore  
This repo is meant to just provide another way of installing TrollInstaller on jailed devices
